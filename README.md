# Sistema7EE2_2
Sistema de integradora

# 🚀 Mi Sistema en Python
Este es un sistema desarrollado en Python utilizando Flask, desplegado en la nube mediante Render.

## 📌 Descripción
Aplicación web que permite en que el usuario pueda solicitar material, en caso de perder o dañar el material pague y verifique como se hace o que tiene que hacer en ocaciones que pueda pagar

## 🛠️ Tecnologías utilizadas
- Python
- Flask
- Gunicorn
- HTML / CSS / JS

## 🌐 Despliegue en Render
El sistema está desplegado en Render y disponible en:
https://Sistema_7EE2_2.com

## ⚠️ Notas
- Asegúrate de tener Python instalado
- El archivo Procfile es necesario para el despliegue en Render
